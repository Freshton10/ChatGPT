### Assisted provisioning for Yealink deskphones

**Before provisioning your Yealink deskphone**

1. Make sure your Yealink phone supports assisted provisioning. 
2. Confirm your deskphone has been assigned a local number or DigitalLine.
3. If you purchased your deskphone from a different provider, check it’s unlocked before you proceed. You can contact the provider to learn if it’s unlocked or how to unlock it.
 
1. Go to the Admin Portal and sign in.
2. Select Phone System in the top menu.
3. Go to Phones & Devices > User Phones. Select a User Phone
4. Select an existing deskphone from the list, then click Setup & Provision.Select Setup and Provision
5. Enter the name of your deskphone brand or model in the search bar, then select it from the dropdown. Select phone.
6. Click Next.
Follow the instructions in the Setup & Provisioning window.
Reset the existing configuration via the web interface.
Press the OK button on the deskphone or press [Menu] > [Status] to get the IP address of the phone.
Open a web browser and enter the deskphone's IP address into the address bar.
Log in to the web interface of the deskphone.
Go to Settings and click Upgrade.
Locate Reset to Factory and click Reset Now.
Reset the existing configuration via the Phone GUI:
On the deskphone, press the OK button for 5 seconds.
Type in the administrator password.
Press Yes to trigger the factory reset.
Note: The web interface will prompt you for admin credentials. By default, it is admin/admin. You may need to contact the phone vendor if you don’t know the credentials.
Enter the deskphone's MAC address in the field.
Click Next. Yealink provisioning flow
Follow the Manual Configuration Instructions in the Finish window. You’ll perform a factory reset on your deskphone, then connect it to RingCentral’s servers.
When you’ve completed all the steps, click Done.
Note: Your deskphone will restart, automatically provisioning to connect to RingCentral’s servers. It will also download any updates and new features.
