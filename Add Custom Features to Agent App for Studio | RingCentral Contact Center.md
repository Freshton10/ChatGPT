### Add Custom Features to Agent App for Studio | RingCentral Contact Center

This article provides steps on how to add custom features to Agent app for Studio under RingCentral Contact Center.
Note: This article does not apply to Engage customers.
 
You can create custom functionality in the agent application through the Indicate action found in Studio.
The Indicate action provides a way to create buttons associated with the agent (agent, team, skill) or contact. When the agent presses these buttons, they can be presented with an HTML form or can signal or spawn a script, run an executable, or open a web page.
The Indicate action is a powerful way to extend the agent application to meet your own custom needs.
