### Directory Sync Overview | RingCentral Contact Center

This article describes the functionality of the Directory Sync feature for RingCentral Contact Center.
Note : This article does not apply to Engage customers.
Directory Sync is a RingCentral Contact Center feature that allows Contact Center agents to consult / transfer calls to any RingCentral MVP extension through the “Corporate Directory” dynamic address book accessible via their agent adapter. It updates the Corporate Directory, through a daily sync (default at 2:00 am US/Mountain Time), to be aligned with the latest RingCentral MVP user lists and is available to any RingCentral Contact Center free of charge. 
User-added image
Directory Sync also supports presence tracking for Corporate Directory entries in agent address books. Agents, when looking up a Corporate Directory entry (e.g. to do a consult / transfer) in their address books, will see a color icon telling whether the entry is available for a call or not:

Gray - Offline
Green - Available
Red - Unavailable/Busy

You see Dynamic type address book named “Corporate Directory”.  You can follow the steps on Address Book - Assign/Unassign to/from a User, Campaign, Team, or Skill if you want to modify assignment of this address book.
