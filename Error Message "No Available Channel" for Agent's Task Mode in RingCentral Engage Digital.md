### Error Message "No Available Channel" for Agent's Task Mode in RingCentral Engage Digital

Agents receive "No Available Channel" error when they try to use Engage Digital's Task Mode.
Digital Engage's Task Mode does not provide aux code features that are always used by agents such as available, break, etc.
Task Mode no channel available
RingCentral Engage Digital
Lack of necessary agent-level permission to access the mentioned feature.
Agents experiencing this issue in RingCentral Engage Digital have not been granted sufficient permissions for the channel sources.
An Admin needs to grant the agents both read and reply permissions for the sources to resolve this concern. Admins need to ensure that the agents receive a corresponding source identity under their respective agent profiles.
For Admins, please see below procedures for granting agent-level permissions:
Log in to RingCentral Engage Digital .
Click the Admin tab on top of the page.
Admin tab
Under Agent Management category, click the Agents button.
Agent button
The agent search window opens, look for the agent by clicking the Search button.
Search agent
Click the " Key " icon at the far right of the page.
permissions icon
Grant the agent both read and reply permissions to the source(s) that they will be supporting.
Add permission to agents.
Click the Save button from the bottom right corner of the page. Return to the agent permission screen like in step number 5.
Select the phone book icon as indicated below:
Agent contact button
Ensure that agents have the corresponding identities to the sources that they will be supporting. See: Configure Agent Identity | RingCentral Engage Digital for more information on how to configure an agent identities.
Click the Save button.
NOTES :
For more details regarding different levels of permission, please read  Configure Agent Permissions | RingCentral Engage Digital .
If the issue persists after completing the above steps please call support at 888-898-4591 or email Engage Digital Support at support.engagedigital@ringcentral.com.
