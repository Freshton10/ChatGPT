### Getting the SIP settings for manual provisioning | RingCentral MVP

The Session Initiation Protocol (SIP) settings allow third-party IP phones to be manually provisioned and connected to the RingCentral server. This will enable the phone to make and receive phone calls.

1. Go to the Admin Portal and sign in.

2. Go to Phone System > Phones & Devices > User Phones.

3. Select Existing Phone. If you donâ€™t have this under your User Phones list, you can add one by clicking Add Device.

4. Click Set Up and Provision.

5. Click the Set up manually using SIP.

6. Click the Set up manually using SIP

7. Select No for Step 1: Will you be using secure voice transport on this device?

For Step 2, click the dropdown menu to choose the appropriate Outbound Proxy based on the location of your device.

Copy the SIP information and click Done to close the window.
 

**Adding an Existing Phone for manual provisioning**

1. Go to the Admin Portal and sign in.

2. Navigate to Phone System > Phones & Devices > User Phones.

3. Click Add Device.

4. Select Domestic, then click Next. To add a desk phone that will be used in an international location, go to Add Global MVP User With a Phone.

5. Select the user to whom you want to assign the phone and click Next.

6. Under Select Devices, go to Other Phones tab and select Existing Phone.

7. go to Other Phones tab

8. Select the number of devices you want to add, click Add, then click Next.

9. Complete the purchase by assigning the number, adding the emergency response location, and selecting the shipping information.

10. Confirm your purchase by ticking the acknowledgment checkbox and clicking Next.

11. Click Done.
