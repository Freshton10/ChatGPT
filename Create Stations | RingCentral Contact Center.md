### Create Stations | RingCentral Contact Center

This article provides instructions on how to create single and multiple Stations.
**NOTE** : This article does not apply to Engage customers.
Administrators have two options when creating Stations:
•    Create a Single Station
•    Create/Edit Multiple Stations
 
Required security profile permission:  Stations Create
 
 
**Create a Single Station**
Go to ACD > ACD Settings  >  Stations .
 
ACD Settings Station
Click  Create New  and select  Single Station .
Single Station Selection
Enter required information in the fields designated with a red asterisk.
User-added image
Click  Create Station .
 
**Create/Edit Multiple Stations**
NOTE:  Best practice is to upload stations in batches of 100 or fewer.
Go to ACD > ACD Settings > Stations .
Click  Create New > Manage Multiple Stations .
To create new stations, click  Download Template . To modify existing stations, or download a list of stations for other uses, click  Download Existing Stations.
User-added image
NOTE:  If you intend to use the list elsewhere, you may disregard the remaining steps in this task.
Open the downloaded file in an application that will let you edit it (e.g.,  Excel or Notepad).
Complete the applicable fields for a new station, or modify the fields you want to change for an existing station.
Save the file.
On the Manage Multiple Stations page, click  Choose File . Repeat the first two steps of this procedure if you no longer have the page open.
Browse for the file you created, select it, and click  Open .
Select the  File Type  corresponding with the file you chose.
Click  Upload File.
The  Data File Processing & Results  page displays a note that you should receive the results of the upload via email. Check your email to verify that no errors occurred during the upload. Resolve any errors that appear by fixing the issue in the file you uploaded, then clicking  Back  and repeating the upload. If you have no errors, click  Close .
**IMPORTANT:**   If the email indicates that the batch failed, you can find details about the processing of the batch files under the  Schedule Run History Report .
