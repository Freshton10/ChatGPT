### Enabling the Report an Issue feature
Engage Voice’s Report an Issue feature lets agents provide feedback about issues with inbound or outbound calls.
Note: The Report an Issue feature does not automatically create support tickets. Customer admins should open RingCentral support tickets if necessary.

To turn on the Report an Issue feature:
1. Log in to Engage Voice, then click Voice Admin.
2. In the left navigation bar, click Settings > Accounts.
3. Click the name of the account you want to edit to expand the menu, then click the relevant subaccount.
4. Scroll down to Agent Settings.
5. Check the box for Enable Agent Call Quality Reporting.
6. Click the Reporting Destination dropdown, then select one of the following options:
Share with RingCentral only
Send an email only
Send an email and share with RingCentral
7. Click Save.
